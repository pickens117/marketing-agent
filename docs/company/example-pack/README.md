# Example Company Pack

This folder shows what a stronger company-context setup can look like for teams adopting this agent.

Suggested files:

- `brand-guidelines.md`
- `message-house.md`
- `personas.md`
- `research-summary.md`
- `legal-constraints.md`
- `company-context.md`

Copy these patterns into `docs/company/` in a real project and tailor them to your company.
