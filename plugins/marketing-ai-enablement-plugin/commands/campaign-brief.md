# Campaign Brief

Create a practical marketing campaign brief for: $ARGUMENTS

Before answering:

- review `docs/company/company-context.md` if it exists
- use any referenced files in `docs/company/` as the source of truth for brand voice, personas, messaging, and constraints
- if company context is missing, say that the brief is generic and label assumptions clearly

The brief should be concise, actionable, and useful for a real marketing team.

Use this structure:

1. Objective
2. Audience
3. Key message
4. Offer or CTA
5. Channel plan
6. Content assets needed
7. Risks and review checkpoints
8. KPIs
9. Recommended next actions

If the request is underspecified, sharpen the brief before proposing tactics.
